# Notix

AI-powered document summarizer for students and professionals. Upload PDFs, Word docs, PowerPoints, and more to get instant summaries, MCQ quizzes, and Q&A chat. Customize output with multiple AI models, summary lengths, tones, and styles. Works offline with history sync. Cross-platform: Windows, macOS, Linux, iOS, Android, and web.
